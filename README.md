## Introduction

This project is a Transportation Management System designed to streamline and manage transportation-related tasks efficiently. It includes Services such as Ticket Booking , Vehicle Booking , Goods & Logistics.

## Features
- Ticket Booking
- Vehicle management
- Driver management
- Route management
- Goods & Logistics
- Scheduling 
- User authentication
- Responsive design


   
## Usage

To use the Transportation Management System, navigate to the project URL in your web browser (e.g., http://localhost:3307/Transporataion-Management-System-php) and log in with the provided credentials or create a new account.
